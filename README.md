# 📊 Blinkit Grocery Data Analysis Project

## 📌 Overview
This project focuses on analyzing Blinkit grocery sales data using Python, Excel, and Power BI.  
It includes data cleaning, feature engineering, and visualization to generate business insights.

---

## 🎯 Objectives
- Clean raw grocery dataset
- Handle missing and invalid values
- Standardize categories
- Create new features (Outlet Age)
- Build Power BI dashboard for insights

---

## 🛠 Tools Used
- Python (Pandas)
- Excel
- Power BI
- Google Colab / VS Code

---

## 📂 Files in Repository
- blinkit_cleaning.py → Python data cleaning script  
- BlinkIT Grocery Data Raw.xlsx → Original dataset  
- cleaned_blinkit_data.xlsx → Cleaned dataset  
- blinkit.pbix → Power BI dashboard file  

---

## 📊 Key Insights
- Tier 3 outlets contribute significant sales  
- Low Fat products are more commonly sold  
- Fruits, Snacks, and Household items are top categories  
- Outlet age impacts performance  

---

## 🚀 Outcome
A complete end-to-end data analysis project from raw data to business insights using Python and Power BI.
